A Sudoku Solver in C++

For this piece of software, we were assigned to create a new C++ Application to solve 6 pre-generated sudoku puzzles. These puzzles could be completed using the naked and hidden singles method of solving sudoku puzzles. This is where you create a "candidate list" for each space in the puzzle, where you have either 1 single value that could be inputted into the space (naked single) or you would be able to create a single number by comparing all the lists in the adjacent 3x3 cube, row and column surrounding that cell and thus having the value to be inputted (hidden single). This is done in C++ by creating an 9x9 array of Cell objects that store the "value" and its candidate list for that value if the value is equal to 0. This is then accessed by another class object that scans the surrounding row, column and 3x3 around the current value and created the candidate list, this is then checked to see if the candidate list only has a single value, if so it is inputted, if not, it will cycle through the remaining cells that are empty until the program solves the sudoku puzzle. The assignment also checked the efficiency of the code i have written, with my program solving the sudoku's in as quickly as 0.007 of a second.
