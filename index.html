Current Project:

Having just finished an internship with the University of Hull's Professor on his personal project threethinggames, i have been working on some small apps and games for friends through lockdown. The internship that i was working on, was to update and add features to an existing full stack web application. This meant that i was working with a ReactJS client (web app) and a Node.JS server with a mongoDB database. All this interfaced with eachother through a custom made API through Express. I added some features that the client wanted adding, such as creating a file upload section to add a custom list of words, the ability to edit an existing event if you were admin and adding a word limit to the software. The majority of these features required me to work on both the client and the server in order to have the function working as intended. I also managed to create a C# app for my friends to do the weakest link style quiz during the lockdown which had the basic functionality of the TV show The Weakest Link.
